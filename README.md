# **Blood Bank Management System**

### Installation

###### Use the package manager [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) to install all prerequisites.
###### Run 
```
npm install
```  
###### or 
```
npm i
```

### Execution
###### 
```
npm start
```
